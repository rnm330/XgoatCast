# XgoatCast · 小羊屏幕共享

聊天软件快捷屏幕共享工具。在频道发条消息即可发起共享，观众免登录观看。当前支持 **KOOK**。

## 功能特性

- **免登录观看**：频道成员点击卡片即可观看，无需安装、无需注册
- **低延迟直播**：支持低延迟直播模式，最低可达 200ms，满足不同场景需求
- **窗口声音隔离**：只共享选中窗口的声音，语音软件不被采集，解决共享时回声问题
- **自动节费管理**：自动检测共享状态，严格管理滥用消耗，控制声网预算，无需手动干预
- **多画质可选**：480p ~ 4K 多档画质自由切换，适应不同网络环境和清晰度需求
- **服务器主自管理**：每服务器独立管理面板，频道主可自行配置画质、Agora 凭证、超时参数

## KOOK 使用指南

### 1. 邀请机器人
邀请机器人到你的 KOOK 服务器。机器人加入后自动向频道主发送绑定卡片。

### 2. 绑定管理面板
频道主点击绑定卡片，设置管理密码，然后配置 Agora App ID 和 App Certificate（每服务器独立配置）。若需重新调起绑定，频道主发送 `/xchelp`。

### 3. 发起屏幕共享
频道内发送触发词（默认「屏幕共享」）→ 机器人推送卡片 → 点击「开始共享」→ 授权屏幕即可。频道成员点击卡片免登录观看。

> 触发词由**超管**在超管面板中为各服务器配置，频道主不可自行修改。

### 可用指令
| 指令 | 说明 |
|------|------|
| `屏幕共享` | 发起共享（默认触发词，超管可在面板自定义） |
| `/xchelp` | 频道主：绑定/管理面板；普通成员：使用说明卡片（含一键发起共享按钮） |

## 部署（服务端）

```bash
cp .env.example .env
# 填入 SUPER_ADMIN_PASSWORD
docker compose up -d --build
```

### 环境变量

| 变量 | 说明 |
|------|------|
| `SUPER_ADMIN_PASSWORD` | **必填**，超管登录密码 |
| `PORT` | 端口，默认 3520 |

> 其余所有配置均在面板操作。

## 模块结构

```
server/src/modules/
├── database/       SQLite 数据层
├── agora/          Agora Token 生成
├── kook/           KOOK Bot 客户端 + 卡片 + 消息处理
├── session/        会话生命周期 + SSE 控制器
├── share/          共享页 API
├── auth/           分享链接鉴权 Guard
├── events/         模块间事件总线
├── super-admin/    超管面板 API
└── server-admin/   服务器管理员 API

web/src/
├── pages/          首页 / 共享 / 观看 / 超管 / 服务器管理
├── hooks/          useScreenShare / useAgoraView / useSessionSSE
└── lib/            API 封装 / 音频劫持 / 工具函数
```
